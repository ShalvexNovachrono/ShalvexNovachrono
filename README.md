### About Me

I am Nava Majumdar, a Computer Games Design and Programming student at the University of Staffordshire. I am passionate about game development, web development, and software engineering.

I build everything from 2D games with pathfinding systems to custom game engines with 3D rendering capabilities. Beyond gaming, I have experience in full-stack web development, creating database systems with PHP and SQL, and contributing to the developer community on Stack Overflow and Replit.
